## üõ†Ô∏è Funzioni Disponibili

MAST Learn offre **23 funzioni** organizzate in 5 categorie principali. Ecco le pi√π importanti che userai:

### üé• **Cattura Immagini** (Le Basi)

**`capture_webcam_image(camera_index=0)`** üì∏
- Scatta una foto con la webcam e la salva automaticamente
- Perfetto per iniziare - funziona subito!
- *Restituisce*: Il percorso dove √® salvata l'immagine

### ü§ñ **Gestione Modelli** (Il Motore)

**`load_custom_model(model_path)`** üß†
- Carica il "cervello" dell'intelligenza artificiale
- Supporta modelli da file locali, percorsi personalizzati o da internet
- *Trucco*: Carica una volta, usa tante volte (molto pi√π veloce!)
- *Bonus*: Converte automaticamente i formati problematici

**Dove mettere i tuoi modelli:**
- **Windows**: `C:\Users\TuoNome\models\mio_modello.h5`
- **Mac**: `/Users/TuoNome/models/mio_modello.h5`
- **Linux**: `/home/TuoNome/models/mio_modello.h5`
- **Personalizzato**: Qualsiasi percorso completo che vuoi!
- **Online**: URL diretti da GitHub o altri server

### ‚ö° **Predizioni Webcam** (La Magia)

**`webcam_predict_label(model, camera_index, class_names)`** üéØ
- Scatta una foto E riconosce l'oggetto in un colpo solo
- *Restituisce*: Il nome dell'oggetto riconosciuto

**`webcam_predict_confidence(model, camera_index, class_names)`** üìä
- Come sopra, ma ti dice quanto √® sicuro (da 0.0 a 1.0)
- *Perfetto per*: Capire se fidarsi della predizione

### üÜï **Predizioni su Immagini Esistenti** (Novit√†!)

**`predict_label_from_image(model, image_path, class_names)`** üîç
- Riconosce oggetti in immagini gi√† salvate
- *Ideale per*: Analizzare la stessa foto pi√π volte

**`predict_confidence_from_image(model, image_path, class_names)`** üìà
- Come sopra, ma restituisce il livello di confidenza
- *Perfetto per*: Confrontare predizioni sulla stessa immagine

### üåê **Condivisione Online** (Per i Social!)

**`send_prediction_data(image_path, label, confidence, api_url)`** üì§
- Invia le tue scoperte a un server web
- *Utile per*: Creare app web o condividere risultati

**`webcam_predict_and_send(model, camera_index, class_names, api_url)`** üöÄ
- Fa tutto in automatico: scatta, riconosce, invia!
- *Perfetto per*: Progetti avanzati o demo live

### üîß **Risoluzione Problemi** (Il Pronto Soccorso)

**`verify_model_compatibility(model_path)`** ü©∫
- Controlla se il tuo modello funziona correttamente
- *Utile quando*: Qualcosa non va e non sai perch√©

**`convert_model_format(input_path, output_path, target_format)`** üîÑ
- Converte modelli tra formati diversi
- *Salva-vita per*: Modelli che danno errori strani
